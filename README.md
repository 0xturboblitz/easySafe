# EasySafe

EasySafe allows you to create a multisig together with non crypto users.

<p align="center">
    <img align="center" src="https://i.imgur.com/w1qWHI2.png" alt="EasySafe Logo" width="400"/>
</p>


EasySafe allows multisig creators to add signers with only an email. An email is then send and the user can login with google or his email through web3auth.
Web3auth creates an address for the user.

<p align="center">
    <img align="center" src="https://i.imgur.com/TtadLKo.png" alt="EasySafe Logo" width="400"/>
</p>

Flexibility of Safes allows the signers to change for more secure addresses when he is more familiar with the tools of crypto.
